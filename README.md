### About

Met dit project intoduceer in een nieuwe manier van matchen in de wereld van standaard matching applicaties.
Dit project zal daarnaast als basis dienen voor een gezamlijk project.

### Features

In dit project ga ik een deel van een applicatie uitwerken voor een matching app. Dit deel zal het koppelen van gebruikers door middel van smaak zijn. Wat de gebruiker invult zal worden verwerkt en als basis voor een match dienen.
